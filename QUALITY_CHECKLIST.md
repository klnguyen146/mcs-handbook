# QUALITY_CHECKLIST.md

> Publication Quality Checklist for the Master Computer Science Handbook (MCSH)

---

# Purpose

This checklist defines the minimum quality requirements that every generated chapter must satisfy before being considered complete.

It serves as a self-review mechanism to ensure consistency, completeness, and academic rigor.

---

# Curriculum Alignment

- [ ] Matches the curriculum mapping.
- [ ] Appropriate for the target Volume.
- [ ] Prerequisites are respected.
- [ ] Learning objectives are achieved.

---

# Educational Quality

- [ ] Clear motivation.
- [ ] Strong intuition.
- [ ] Smooth learning progression.
- [ ] Graduate-level explanations.

---

# Technical Accuracy

- [ ] Definitions are correct.
- [ ] Mathematics is accurate.
- [ ] Algorithms are correct.
- [ ] Complexity analysis is provided where appropriate.
- [ ] Engineering discussions are technically sound.

---

# Visual Quality

- [ ] Relevant figures included.
- [ ] Tables improve understanding.
- [ ] Diagrams are referenced in the text.
- [ ] Visuals are educational rather than decorative.

---

# Practical Content

- [ ] Code examples.
- [ ] Worked examples.
- [ ] Real-world applications.
- [ ] Engineering insights.
- [ ] Common mistakes.

---

# Learning Support

- [ ] Summary.
- [ ] Key takeaways.
- [ ] Exercises.
- [ ] Mini project.
- [ ] Further reading.

---

# Academic References

- [ ] Correct terminology.
- [ ] Landmark papers referenced when appropriate.
- [ ] Canonical textbooks recommended.
- [ ] Historical context included where valuable.

---

# Language Quality

- [ ] Main content in Vietnamese.
- [ ] Important technical terms remain in English.
- [ ] Consistent terminology.
- [ ] Clear and concise writing.

---

# Publication Readiness

The chapter is:

- [ ] Complete
- [ ] Consistent
- [ ] Accurate
- [ ] Readable
- [ ] Educational
- [ ] Publication-ready

---

# Final Decision

Only publish the chapter if all mandatory criteria are satisfied.

If significant issues remain, revise before generating the final version.
